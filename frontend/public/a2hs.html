Add to Home Screen

This Progressive Web App (PWA) can be installed to your device so you can open it like a native app.

Install instructions

  • Android (Chrome/Edge): Open the browser menu (⋮) and tap "Add to Home screen" or tap the install icon in the address bar.
  • iOS (Safari): Tap the Share button and choose "Add to Home Screen". Note: iOS does not support the automatic install prompt.
  • Desktop (Chromium browsers): Use the install prompt in the address bar or the browser menu -> "Install app".

Notes

  • After installation the app runs in a standalone window without browser UI.
